Summary

1. Web page is a text document written in a language called HTML. Web browsers read these documents and then interpret and display them.

2. Coding happens when programmers write text in a language that a computer can understand. The computer can then follow the instructions the programmer wrote.

3. Computers are stupid. Programmers need to write exactly the way a computer understands aka writing with correct syntax.

4. Programmers can’t remember everything. Can always look up in google.

5. Basic HTML vocabulary.

Tag-always contained within angled brackets. Most have an opening tag and a closing tag.

Void tags-don’t require a closing tag

Element-an HTML element refers to everything within a set of opening and closing tags.

Attribute-a property of an HTML element. e.g. to set the HREF attribute of an anchor tag to the Udacity URL, you would write URL
